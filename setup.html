HOW TO USE

Setup

Adding Axora to Your Discord Server

Getting started with Axora is simple. Follow these steps to add the bot to your Discord server:

1

Visit the Invite Link

Go to Axora's invite page

2

Select Your Server

Choose the server where you want to add Axora from the dropdown menu

3

Authorize the Bot

Review the permissions and click "Authorize"

4

Complete Verification

Complete any CAPTCHA verification if prompted

Initial Configuration

Once Axora is in your server, you'll want to configure it for optimal use:

Basic Commands 
/setup - Run the initial setup wizard
/config - Access configuration options
/help - View all available commands

Note: Axora requires the "Administrator" permission to function properly. If you're concerned about permissions, you can create a dedicated role with limited access.
