HOW TO USE

Setup

Getting started with Axora is quick and easy. This guide will walk you through the installation process and initial configuration.

System Requirements

  • Operating System: Windows 10/11, macOS 10.15+, or Linux (Ubuntu 18.04+)
  • Memory: Minimum 4GB RAM (8GB recommended)
  • Storage: 500MB free space
  • Internet: Broadband connection

Installation

# Download the installer
curl -o axora-setup.exe https://download.axora.ai/latest/windows

# Run the installer and follow the on-screen instructions
./axora-setup.exe

Alternatively, you can download the installer from our official website.

# Using Homebrew
brew install axora

# Or download the .dmg file
curl -o axora.dmg https://download.axora.ai/latest/macos
# Using apt (Ubuntu/Debian)
sudo apt update
sudo apt install axora

# Using snap
sudo snap install axora
# Pull the Docker image
docker pull axora/axora:latest

# Run the container
docker run -d -p 3000:3000 --name axora axora/axora:latest

Initial Configuration

After installation, you'll need to configure Axora with your API key and preferences:

  1. Create an Account

    Visit dashboard.axora.ai to create an account and generate your API key.

  2. Configure API Key

    axora config --api-key YOUR_API_KEY

  3. Verify Installation

    axora verify

    You should see a success message confirming that Axora is properly installed and configured.

Note: For enterprise installations or custom deployments, please refer to our Enterprise Guide.
